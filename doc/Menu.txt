🧱 Global Layout (applies to ALL screens)

Every screen uses this frame:

┌──────────────────────────────────────────────────────────────────────┐
│ MiniOS Setup                                   VT520 Mode  80x24     │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  [ CONTENT AREA ]                                                    │
│                                                                      │
│                                                                      │
│                                                                      │
├──────────────────────────────────────────────────────────────────────┤
│ Status: <dynamic message>                                            │
├──────────────────────────────────────────────────────────────────────┤
│ F1 Help   F2 Back   F3 Select   F4 Edit   F5 Refresh   F10 Quit       │
└──────────────────────────────────────────────────────────────────────┘
Rules:
Arrow keys = navigation
Enter = select
ESC / F2 = back
F10 = exit setup
Reverse video = selected item
No mouse ever
